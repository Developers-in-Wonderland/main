## 논의 내용

### 1️⃣ 평가지표 비교 정리

* 기존 대비 **측정 절차·기준 명확화** 중심으로 수정됨
* 1번은 **표현 보완** 위주(측정 방식 동일)
* 2·3번은 **실험 조건(거리·시간·기준)** 을 구체화하고 **현실적 오차 반영**
* 수정 이유는 “측정자 간 오차 최소화” 및 “재현성 강화”로 정리

---

### 2️⃣ 평가지표 3 실험 결과

* 로봇팔이 **정지 상태에서는 안정적으로 유지**됨 (원 내부 비율 ≥ 80%)
* 하지만 **움직이는 중 발생하는 미세 떨림**이 여전히 존재함
* 문제 원인 분석 결과:

  * 기존 **아날로그 서보모터 → 디지털 서보모터**로 교체 후,
    동일한 PWM 제어값을 적용하자 **1틱 단위 미세 진동**이 과도하게 발생
  * 디지털 모터의 응답 속도와 토크 특성이 달라 **기존 제어 주기와 부조화**

---

### 3️⃣ 개선 방향

| 구분                 | 개선 내용                                | 목적                |
| ------------------ | ------------------------------------ | ----------------- |
| **모터 제어 신호 조정**    | 모터 속도에 따라 PWM 변화량(Δpulse)을 비선형적으로 조정 | 미세 진동 완화          |
| **속도별 제어 프로파일 작성** | 저속·중속·고속 구간별 제어 스텝 차등 적용             | 부드러운 가속/감속 구현     |
| **테스트 루틴 추가**      | 각 축별 응답 로그 측정 및 제어량 보정               | 디지털 서보 특성에 맞는 최적화 |

## 최종 평가지표(2025-10-25 기준)

| 구분                             | 측정 방법                                                           | 목표 지표 |
| ------------------------------ | --------------------------------------------------------------- | ----- |
| **평가지표 1. 얼굴 재인식 시간 (FRT)**    | 이동 방향(수평·수직·대각선) 중 1개 선택 → 2초 이내 이동 → 얼굴이 다시 인식되는 첫 프레임까지 시간 측정 | ≤ 2초  |
| **평가지표 2. 로봇팔 추적 안정성 (IFR20)** | 사용자가 30 cm를 3초 동안 단일 방향으로 이동 → 프레임 간 이동 벡터 ≤ 20 px 비율 계산        | ≥ 80% |
| **평가지표 3. 정지 안정성 (ICR n%)**    | 30 cm 이동 후 3초 정지 → 첫 좌표 중심으로 원(n%) 내부 비율 계산                     | ≥ 80% |
